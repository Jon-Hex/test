# Итоговое задание Python! Анализ сетевого трафика и VirusTotal

Программа на Python для анализа сетевых инцидентов и автоматической проверки подозрительных IP-адресов.

itog.py - программа ДЗ.
- Проверяем дамп сети (alerts-only.json).
- Определяем "подозрительные IP-адреса".
- Выводим диаграммы.
- Отбираем 2 самых подозрительных.
- Отправляем alerts-only.json на проверку в Virus Total.
- Отправляем на проверку 2 самых подозрительных IP-адреса.
- Формируем отчеты в формате .json и .csv.

alerts-only.json — исходные данные.
full_analysis_report.png и threats_pie_chart.png — графики.
vt_results.csv и vt_results.json — итоговые отчеты.
Pict-1.jpg, Pict-2.jpg - Диаграммы.
Pict-3.jpg - итог работы программы в терминале.

## Функционал
* Анализ логов в формате JSON.
* Визуализация угроз (круговая и горизонтальная диаграммы).
* Фильтрация инцидентов с потенциальной утечкой данных.
* Автоматическая проверка IP-адресов через **VirusTotal API**.
* Сохранение отчетов в форматах **CSV** и **JSON**.

## Как запустить
1. Установите зависимости: `pip install pandas matplotlib vt-py`
2. Вставьте свой API-ключ в файл `itog.py`, а лучше создайте переменную окружения "VT_API_KEY"
3. Запустите скрипт: `python itog.py`
